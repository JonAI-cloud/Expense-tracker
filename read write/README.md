# Expense Tracker

A command-line expense tracker built in python.

## Features
- Add, view, delete, and sort expenses
- Categorizes expenses as small, medium, or large
- Filter expenses by category
- Monthly budget tracking with warnings
- Data persists between runs using CSV

## How to run
1. Create a virtual environment: `python -m venv .venv`
2. Activate it: `.venv\Scripts\activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `python expenses.py`
